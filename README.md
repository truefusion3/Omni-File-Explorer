# Omni File Explorer

> The prefix "*Omni-*" is derived from the latin phrase "*Omnis*", meaning "all," "every," "the whole," or "of every kind".

Omni File Explorer is created primarily for academic purposes, but is also created in response to the decay in quality of the search system in MacOS Tahoe.

Primary Tasks:
- [ ] Search files and list them without throwing an error
- [ ] DEV: Create report of succesful and failed searches

Optional: 
- [ ] Create shortcut for easy access
- [ ] Search algorithm toggle (Breadth vs Depth)
- [ ] Priority searching based on keywords
- [ ] AI input integration (GPT, Claude, etc.)

End Goal: 
- [ ] Custom file handling based on filetype
